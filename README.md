# python_script

## aliyun
* 批量创建 阿里云EC2 ，指定镜像、vpc、密钥、硬盘大小、EIP。
* 创建新EIP同时绑定新建EC2主机。

## cloudflare_add_domain
* 批量向cloudflare 添加域名，同时添加域名解析。同时对接了私有cdn和第三方cdn，实现批量添加域名，并解析，自动申请ssl免费证书。

## for_jenkins
* 查看每个项目的commit 信息。

## tk_cloudflare
* 使用tk gui 完成对cloudflare 上域名的CRUD，批量添加域名，根据自定义前缀添加域名解析。
